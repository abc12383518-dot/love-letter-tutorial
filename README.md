- 🧠 5 個基本策略提示
- ❓ 常見問題 Q&A
- 🎨 中世紀童話宮廷風格設計

## 📖 關於情書

《情書》(Love Letter) 是一款經典小品桌遊，2-4 人遊戲，每局約 5-10 分鐘。玩家扮演宮廷中的追求者，利用 16 張角色牌的能力，淘汰對手或在回合結束時持有最大數字的牌來贏得公主的芳心。

## 📄 授權

本教學簡報僅供教育使用。情書 (Love Letter) 桌遊版權屬於原作者 Seiji Kanai 及 AEG。
